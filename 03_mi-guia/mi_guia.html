Introducción al lenguaje

HTML significa HyperText Markup Language
Es el lenguaje basico que se usa para crear y estructurar paguinas web

Etiquetas de Formato

br: etiqueta de HTML que sirve para hacer un salto de línea dentro de un texto. hr: etiqueta de HTML que sirve para crear una línea horizontal que separa contenido o secciones.
